HTML Tips

Headers

Headers can be called using the h1,h2,h3, ... tags. Lower numbers are more important. Bigger numbers imply subsections

Formatting

We also use tags to imply bold, italics, code, etc....

More examples of formatting tags can be found at the following website.

http://www.w3schools.com/html/html_formatting.asp

Hyperlinks

The a tag can be used to link to external websites like Yahoo!, Google, or Wikipedia's article on HTML.

It can also be used to link to local websites like The HelloWorld example of this tutorial.

For more information about links see this page

Images

Images are handled very similarly to links. To the right we've included two images of Tim Berners-Lee, the originator of the term HTML. The rightmost image is served from the wikipedia servers, the left image is served from this computer.

We specify the source location of the image using the "src" attribute. The src can be any URL, usually either a link to an image on the web or a file on our own computer. I've downloaded this image and saved it locally in images/tim.jpg.

There are a few other attributes present. "alt" is the alternative text for displays that don't handle images. "width" and "align" give control about image size and placement. A list of other handy image attributes are available here
