HTML Tips

Headers

Headers can be called using the h1,h2,h3, ... tags. Lower numbers are more important. Bigger numbers imply subsections

Formatting

We also use tags to imply bold, italics, code, etc....

Hyperlinks

In the last section we listed a URL of a webpage that held more information about the topic. A great thing about HTML is that we can embed these links directly into the text. This is done with the <a> tag and by specifying an href attribute as follows.

The text : <a href="http://www.google.com">Google</a> yields Google

In this example the <a></a> tag surrounds the text "Google" and is given the attribute href with value "http://www.google.com"

For more information about links see this page

Images

Images are handled very similarly to links. Standard syntax to include an image of Tim Berners-Lee, originator of the term HTML, looks like the folowing:

<img src="http://upload.wikimedia.org/wikipedia/commons/c/c8/Tim_Berners-Lee_April_2009.jpg" alt="Tim Berners-Lee" width=20% align="right"/>

We specify the source location of the image using the "src" attribute. In this case the source is a URL linking to a wikipedia image. We can also have src point to something local in our directory. I've downloaded this image and saved it locally in images/tim.jpg. We can rewrite the image element as follows:

<img src="images/tim.jpg" alt="Tim Berners-Lee" width=20% align="right"/>
